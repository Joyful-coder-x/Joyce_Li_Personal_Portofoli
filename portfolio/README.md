## How to Use
Each file contains the HTML for a single portfolio item or modal. You can:
- Edit individual projects independently
- Add new project files by following the naming convention
- Reference these files in the main index.html when needed

## Integration Notes
To use these files in your main index.html, you would typically include them using:
- Server-side includes (PHP, Node.js, etc.)
- JavaScript fetch or load methods
- Build tools that concatenate files during deployment
